## Crafting Metadata

Crafting metadata will depend on your language's handling of JSON. Common metadata are:
<UnorderedList items={["Invoice ID", "Cart ID", "Cart Items", "Payment medium (site/app)"]} />

The `custom_fields` key is reserved for an array of custom fields that should show on the dashboard when you click the transaction. 

Custom fields have 3 keys: `display_name`, `variable_name`, and `value`. The display name will be the label for the value when displaying.
